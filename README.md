## デモの開発環境構築

デモコードはTypeScriptで書かれているので、実際に編集や実行するためには環境構築が必要です。以下にそのフローを記載します。

### 1. npmモジュールをインストール

以下のコードを実行して必要なnpmモジュールをインストールしてください。

```bash
npm install
```

### 2. ソースコードのWatchと簡易Webサーバーの起動

以下のコマンドを実行することで、TypeScriptコードのWatchが開始され、編集後に自動的にコンパイルされます。

```bash
npm dev
```
